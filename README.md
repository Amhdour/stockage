# Merchandise Stock App

A simple browser-based inventory app for tracking merchandise stock.

## Features
- Add or update products by SKU
- Track category, price, quantity, and reorder threshold
- Low-stock status highlighting
- Search by name, SKU, or category
- Quick quantity adjustment (+1 / -1)
- Persistent storage in browser `localStorage`

## Run
Open `index.html` in a web browser.

For a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
