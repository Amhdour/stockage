# Stockage: Mobile-Style Merchandise Inventory

A browser-based merchandise stock app redesigned with a mobile-commerce inspired interface.

## Features
- Two mobile-style preview panels inspired by modern e-commerce product + checkout flows
- Add/update inventory by SKU
- Search and sort inventory
- Low-stock status highlighting
- Quick quantity buttons (+1 / -1)
- Summary metrics (products, units, low stock, total value)
- Export inventory JSON
- Browser `localStorage` persistence

## Run
Open `index.html` in a browser.

Or run a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
